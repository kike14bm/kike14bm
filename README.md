### 🏀 Modelo Predictivo NBA 🇺🇸
<p align="left">
  <img src="https://github.com/kike14bm/ModeloPredictivo_NBA/blob/master/images/CurryTiros.png?raw=true" alt="Captura del Modelo Predictivo NBA" width="800" />
